JavaFX Animation


For my final project in Computer Science II, I wrote a program that is capable of executing a few different sources of animation at the same time. This project stresses the capabilites of object oriented programming in the parent-child relationships of classes, overriding and overloading methods, and the use of key object oriented themes such as the super keyword and abstract classes as well as interfaces. This program was written in Java, and the source code for this project can be found on my GitHub account.
