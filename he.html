Huffman Encoding


This is a project from my Data Structures course, written in Java. This program uses a file from the command line, parses the data within the file, and encodes the characters A-G using the Huffman algorithm and binary tree. The user has the options to view a printout of the tree, the table that shows the binary encoding, the encoded portion of the file, and the original decoded version of the file. The source code for this project can be found on my Github account.
