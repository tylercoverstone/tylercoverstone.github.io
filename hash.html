Hashing and Horner's Polynomial


This program takes three data files from the command line to create an array of hashed strings using Horner's Polynomial as a hash function. Each line of the first file is read as a string, and once hashed the string is inserted into the array through both linear and quadratic probing methods. The probing length for each method is averaged and shown so that the results can be observed. Strings for the second file will be searched for in the hash array and the third file will contain strings that the program will delete from the array. This program was written in Java, and the source code can be found on my Github account.
