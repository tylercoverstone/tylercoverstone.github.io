Graph Operations


This program takes a file from the command line that will read characters and insert them into an adjacency list as vertices in a graph, and two charatcers in succession will result in an edge. The adjacency list will then be printed, as well as the results of a depth-first search, breadth-first search, and topological-sort search/traversal methods. This program utilizes strucutres and in-depth pointer utilization, as the adjacency list consists of an array of vertices in which each vertex contains a linked list of edges that contain pointers to other vertices in the array. This program was written in C, and the source code can be found on my Github account.
